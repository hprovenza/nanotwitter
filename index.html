Welcome to Nanotwitter.

nanoTwitter(nT) is a project for the class CS105b - Software Engineering for Scalability at Brandeis Univerisity. It is a baby version of Twitter designed for the purpose of experimenting with scaling issues.

Our project is currently performing the best in our class under load testing with loader.io.
